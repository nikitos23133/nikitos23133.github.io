Урок 4: Работа с массивами и хэшами - Ruby

Введение

Массивы (`Array`) и хэши (`Hash`) — ключевые структуры данных в Ruby для хранения и обработки наборов данных. Массивы хранят упорядоченные списки, а хэши — пары "ключ-значение". В этом уроке вы научитесь создавать, изменять и использовать массивы и хэши.

Работа с массивами

Массив — это упорядоченный список элементов. Создайте файл arrays.rb с кодом:

fruits = ["Яблоко", "Банан", "Апельсин"]
puts fruits # Вывод: Яблоко\nБанан\nАпельсин

# Добавление элемента
fruits << "Груша"
puts fruits # Вывод: Яблоко\nБанан\nАпельсин\nГруша

# Доступ и изменение
puts fruits[1] # Вывод: Банан
fruits[2] = "Мандарин"
puts fruits[2] # Вывод: Мандарин

# Итерация
fruits.each do |fruit|
    puts "Фрукт: #{fruit}"
end

Разберём код:

  • fruits = [...]: Создание массива.
  • <<: Добавление элемента в конец.
  • fruits[1]: Доступ по индексу.
  • each: Перебор элементов.

Работа с хэшами

Хэш — это набор пар ключ-значение. Попробуйте код в hashes.rb:

scores = { "Алексей" => 85, "Мария" => 92, "Иван" => 78 }
puts scores # Вывод: {"Алексей"=>85, "Мария"=>92, "Иван"=>78}

# Добавление/обновление
scores["Ольга"] = 95
puts scores["Ольга"] # Вывод: 95

# Доступ
puts scores["Мария"] # Вывод: 92

# Итерация
scores.each do |name, score|
    puts "#{name}: #{score}"
end

Разберём код:

  • { ... => ... }: Создание хэша.
  • scores["Ольга"]: Добавление/обновление пары.
  • scores["Мария"]: Доступ по ключу.
  • each: Перебор пар.

Запуск кода

Для выполнения кода:

  1. Создайте файл (arrays.rb или hashes.rb).
  2. В терминале выполните: 
    ruby arrays.rb
  3. Или используйте IRB для тестирования кода.

Практика

Выполните задания:

  1. Создайте массив с названиями 5 фильмов и выведите их.
  2. Добавьте новый фильм и замените третий.
  3. Создайте хэш с 3 именами и их баллами.
  4. Обновите баллы одного человека и добавьте нового.
  5. Напишите программу, выводящую имена и баллы в формате "Имя: Баллы".
  6. Создайте массив чисел и выведите только числа больше 50.
  7. Создайте хэш с продуктами и их количеством, затем выведите продукт с наибольшим количеством.

Урок 5: Модули и классы - Ruby

Введение

Модули и классы в Ruby позволяют организовывать код и реализовывать объектно-ориентированное программирование (ООП). Модули группируют методы и константы для повторного использования, а классы определяют объекты с данными и поведением. В этом уроке вы научитесь создавать модули для общей функциональности и классы для моделирования сущностей, таких как студенты или книги.

Работа с модулями

Модули используются для группировки методов или как примеси (mixins) для добавления функциональности в классы. Создайте файл modules.rb с кодом:

module Greeter
  def greet
    "Привет, #{self.name}!"
  end
end

class Person
  include Greeter
  attr_accessor :name

  def initialize(name)
    @name = name
  end
end

person = Person.new("Алексей")
puts person.name      # Вывод: Алексей
puts person.greet     # Вывод: Привет, Алексей!

Разберём код:

  • module Greeter: Определение модуля с методом greet.
  • include Greeter: Включение модуля в класс, добавляя его методы.
  • attr_accessor :name: Создание геттера и сеттера для атрибута name.
  • initialize: Конструктор для инициализации объекта.

Модули также могут содержать константы:

module Constants
  PI = 3.14159
end

puts Constants::PI    # Вывод: 3.14159

Работа с классами

Классы определяют шаблон для объектов. Попробуйте код в classes.rb:

class Student
  attr_accessor :name, :grades

  def initialize(name, grades)
    @name = name
    @grades = grades
  end

  def average_grade
    @grades.sum / @grades.length.to_f
  end
end

student = Student.new("Мария", [85, 90, 78])
puts student.name           # Вывод: Мария
puts student.average_grade  # Вывод: 84.33333333333333
student.grades << 95
puts student.grades         # Вывод: [85, 90, 78, 95]

Разберём код:

  • class Student: Определение класса.
  • attr_accessor: Автоматическое создание методов для чтения/записи атрибутов.
  • initialize: Инициализация объекта с именем и оценками.
  • average_grade: Метод для вычисления среднего балла.

Запуск кода

Для выполнения кода:

  1. Создайте файл (modules.rb или classes.rb).
  2. Откройте терминал в папке с файлом.
  3. Выполните: 
    ruby modules.rb
  4. Или используйте IRB для тестирования: 
    irb
    Скопируйте код и проверьте вывод.

Убедитесь, что Ruby установлен (ruby -v).

Практика

Выполните следующие задания для закрепления:

  1. Создайте модуль MathUtils с методом для вычисления квадрата числа. Используйте его в классе.
  2. Создайте класс Book с атрибутами title и year. Добавьте метод description для вывода информации о книге.
  3. Создайте экземпляр Book и выведите его описание.
  4. Создайте модуль Speed с методом accelerate, увеличивающим скорость. Включите его в класс Car.
  5. Создайте класс Team с атрибутом members (массив имён) и методом для вывода их количества.
  6. Напишите программу, использующую модуль и класс для моделирования сотрудника с именем и зарплатой, включая метод для повышения зарплаты.
  7. Создайте класс Rectangle с атрибутами width и height. Добавьте метод area для вычисления площади.

Подробности о модулях и классах читайте в документации Ruby.

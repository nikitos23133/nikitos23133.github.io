Урок 2: Переменные, типы данных и операторы - Ruby

Введение

Ruby — это динамически типизированный язык, в котором переменные не требуют явного объявления типов. В этом уроке вы изучите переменные, основные типы данных и операторы в Ruby, которые позволяют выполнять вычисления и манипуляции с данными.

Переменные

В Ruby переменные создаются простым присваиванием значения. Тип данных определяется автоматически.

name = "Алиса"
age = 25
puts "Имя: #{name}, Возраст: #{age}"

В этом примере переменные name (строка) и age (целое число) используются для вывода текста.

Типы данных

Ruby поддерживает несколько основных типов данных:

  • Integer: Целые числа (например, 42).
  • Float: Числа с плавающей точкой (например, 3.14).
  • String: Строки (например, "Hello").
  • Boolean: Логические значения (true, false).
  • Array: Массивы (например, [1, 2, 3]).
  • Hash: Ассоциативные массивы (например, { name: "Алиса", age: 25 }).
number = 42
pi = 3.14
greeting = "Привет!"
is_active = true
numbers = [1, 2, 3]
person = { name: "Боб", age: 30 }

puts number, pi, greeting, is_active, numbers, person

Операторы

Операторы в Ruby позволяют выполнять вычисления и сравнения.

Арифметические операторы

a = 10
b = 3
puts a + b  # 13
puts a - b  # 7
puts a * b  # 30
puts a / b  # 3
puts a % b  # 1
puts a ** 2 # 100

Операторы сравнения

x = 5
y = 3
puts x == y  # false
puts x != y  # true
puts x > y   # true
puts x <= y  # false

Логические операторы

is_adult = true
has_ticket = false
puts is_adult && has_ticket  # false
puts is_adult || has_ticket  # true
puts !is_adult               # false

Практика

1. Создайте программу, которая объявляет переменные для имени, возраста и города, и выводит их.

2. Напишите программу, выполняющую арифметические операции с двумя числами (сложение, деление).

3. Создайте массив чисел и выведите его элементы.

4. Напишите программу, использующую хэш для хранения данных о человеке и выводящую их.

5. Проверьте, является ли число положительным, используя операторы сравнения.

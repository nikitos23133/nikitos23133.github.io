Урок 2: Переменные, типы данных и операторы - Swift

Введение

Swift — это строго типизированный язык с поддержкой вывода типов. В этом уроке вы изучите переменные, константы, основные типы данных и операторы в Swift для выполнения вычислений и манипуляций с данными.

Переменные и константы

Swift использует var для переменных и let для констант.

var name = "Алиса"
let age = 25
print("Имя: \(name), Возраст: \(age)")

Константы (let) нельзя изменять после инициализации.

Типы данных

Swift поддерживает следующие основные типы данных:

  • Int, UInt: Целые числа (например, 42).
  • Double, Float: Числа с плавающей точкой (например, 3.14).
  • String: Строки (например, "Hello").
  • Bool: Логические значения (true, false).
  • [Type]: Массивы (например, [1, 2, 3]).
let number = 42
let pi = 3.14
let greeting = "Привет!"
let isActive = true
let numbers = [1, 2, 3]
print(number, pi, greeting, isActive, numbers)

Операторы

Операторы в Swift позволяют выполнять вычисления и сравнения.

Арифметические операторы

let a = 10
let b = 3
print(a + b) // 13
print(a - b) // 7
print(a * b) // 30
print(a / b) // 3
print(a % b) // 1

Операторы сравнения

let x = 5
let y = 3
print(x == y) // false
print(x != y) // true
print(x > y)  // true
print(x <= y) // false

Логические операторы

let isAdult = true
let hasTicket = false
print(isAdult && hasTicket) // false
print(isAdult || hasTicket) // true
print(!isAdult)            // false

Практика

1. Создайте программу, которая объявляет переменные и константы для имени, возраста и роста, и выводит их.

2. Напишите программу, выполняющую арифметические операции с двумя числами (умножение, деление).

3. Создайте массив чисел и выведите его элементы.

4. Напишите программу, использующую переменные разных типов (Int, Double, String) и выводящую их.

5. Проверьте, является ли число чётным, используя оператор %.

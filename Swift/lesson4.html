Урок 4: Работа с массивами и словарями - Swift

Введение

Массивы и словари — ключевые структуры данных в Swift, позволяющие хранить и обрабатывать наборы значений. Массивы хранят упорядоченные списки элементов, а словари — пары "ключ-значение". В этом уроке вы научитесь создавать, изменять и использовать массивы и словари в Swift. Эти навыки пригодятся для разработки iOS/macOS приложений.

Работа с массивами

Массив (`Array`) в Swift — это упорядоченная коллекция элементов одного типа. Создайте файл arrays.swift и попробуйте следующий код:

var fruits: [String] = ["Яблоко", "Банан", "Апельсин"]
print(fruits) // Вывод: ["Яблоко", "Банан", "Апельсин"]

// Добавление элемента
fruits.append("Груша")
print(fruits) // Вывод: ["Яблоко", "Банан", "Апельсин", "Груша"]

// Доступ к элементу
print(fruits[1]) // Вывод: Банан

// Изменение элемента
fruits[2] = "Мандарин"
print(fruits) // Вывод: ["Яблоко", "Банан", "Мандарин", "Груша"]

// Итерация по массиву
for fruit in fruits {
    print("Фрукт: \(fruit)")
}

Разберём код:

  • var fruits: [String]: Объявление массива строк.
  • append: Добавляет элемент в конец массива.
  • fruits[1]: Доступ к элементу по индексу (начинается с 0).
  • for ... in: Цикл для перебора элементов.

Работа со словарями

Словарь (`Dictionary`) хранит пары "ключ-значение", где ключи уникальны. Попробуйте следующий код в dictionaries.swift:

var scores: [String: Int] = ["Алексей": 85, "Мария": 92, "Иван": 78]
print(scores) // Вывод: ["Иван": 78, "Алексей": 85, "Мария": 92]

// Добавление/обновление элемента
scores["Ольга"] = 95
print(scores) // Вывод: ["Иван": 78, "Алексей": 85, "Мария": 92, "Ольга": 95]

// Доступ к значению
if let score = scores["Мария"] {
    print("Оценка Марии: \(score)") // Вывод: Оценка Марии: 92
}

// Итерация по словарю
for (name, score) in scores {
    print("\(name): \(score)")
}

Разберём код:

  • var scores: [String: Int]: Словарь с ключами-строками и значениями-целыми числами.
  • scores["Ольга"] = 95: Добавление или обновление пары.
  • if let: Безопасный доступ к значению по ключу.
  • for (name, score) in: Перебор пар ключ-значение.

Запуск кода

Для тестирования кода используйте Xcode Playground или создайте Swift-файл:

  1. Откройте Xcode и создайте новый Playground или проект.
  2. Скопируйте код из примеров.
  3. Запустите (Cmd+R) и проверьте вывод в консоли.
  4. Для файлов (arrays.swift, dictionaries.swift) используйте терминал: 
    swift arrays.swift

Практика

Выполните следующие задания для закрепления:

  1. Создайте массив с названиями 5 ваших любимых книг и выведите их.
  2. Добавьте новую книгу в массив и замените вторую книгу на другую.
  3. Создайте словарь с названиями 3 предметов и их оценками (например, "Математика": 90).
  4. Обновите оценку одного предмета и добавьте новый предмет.
  5. Напишите программу, которая выводит все предметы и оценки из словаря в формате "Предмет: Оценка".
  6. Создайте массив чисел и выведите только чётные числа.
  7. Создайте словарь с днями недели и количеством задач (например, "Понедельник": 5), затем выведите день с наибольшим количеством задач.

Урок 5: Пакеты и модули - Go

Введение

Пакеты и модули в Go позволяют организовывать код для повторного использования и масштабируемости. Пакеты группируют связанные функции и типы, а модули управляют зависимостями. В этом уроке вы научитесь создавать пакеты, использовать структуры (`struct`) и работать с модулями.

Создание пакета

Пакет — это каталог с Go-файлами. Создайте структуру директорий:

myapp/
├── main.go
└── person/
    └── person.go

В person/person.go добавьте код:

package person

type Person struct {
    Name string
    Age  int
}

func (p Person) Greet() string {
    return fmt.Sprintf("Привет, я %s!", p.Name)
}

В main.go используйте пакет:

package main

import (
    "fmt"
    "myapp/person"
)

func main() {
    p := person.Person{Name: "Алексей", Age: 25}
    fmt.Println(p.Name) // Вывод: Алексей
    fmt.Println(p.Greet()) // Вывод: Привет, я Алексей!
}

Разберём код:

  • package person: Объявление пакета.
  • type Person struct: Структура с полями.
  • Greet: Метод структуры.
  • import "myapp/person": Импорт пакета.

Работа с модулями

Модули управляют зависимостями. Инициализируйте модуль:

  1. В корне проекта выполните: 
    go mod init myapp
  2. Запустите программу: 
    go run .

Для использования внешних пакетов добавьте их с помощью go get.

Запуск кода

Для выполнения:

  2. Инициализируйте модуль (go mod init myapp).

Практика

Выполните задания:

  1. Создайте пакет book с структурой Book (поля Title, Year) и методом Description.
  2. Используйте пакет book в main.go для вывода описания книги.
  3. Создайте пакет car с структурой Car и методом Accelerate.
  4. Инициализируйте модуль и запустите программу с использованием пакета car.
  5. Создайте пакет point с структурой Point (координаты) и методом Distance.
  6. Создайте пакет team с структурой Team и методом для подсчёта участников.
  7. Напишите программу, использующую два пакета для моделирования студентов и их оценок.

Подробности о пакетах читайте в документации Go.

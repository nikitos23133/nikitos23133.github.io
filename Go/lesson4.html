Урок 4: Работа со срезами и картами - Go

Введение

Срезы (`slice`) и карты (`map`) — основные структуры данных в Go для работы с наборами элементов. Срезы представляют динамические массивы, а карты — ассоциативные массивы (ключ-значение). В этом уроке вы научитесь создавать, изменять и использовать срезы и карты для эффективной обработки данных.

Работа со срезами

Срез — это динамическая обёртка над массивом. Создайте файл slices.go с кодом:

package main

import "fmt"

func main() {
    fruits := []string{"Яблоко", "Банан", "Апельсин"}
    fmt.Println(fruits) // Вывод: [Яблоко Банан Апельсин]

    // Добавление элемента
    fruits = append(fruits, "Груша")
    fmt.Println(fruits) // Вывод: [Яблоко Банан Апельсин Груша]

    // Доступ и изменение
    fruits[1] = "Манго"
    fmt.Println(fruits[1]) // Вывод: Манго

    // Итерация
    for _, fruit := range fruits {
        fmt.Println("Фрукт:", fruit)
    }
}

Разберём код:

  • []string: Объявление среза строк.
  • append: Добавляет элемент в конец среза.
  • fruits[1]: Доступ по индексу.
  • for ... range: Перебор элементов.

Работа с картами

Карта — это набор пар ключ-значение. Попробуйте код в maps.go:

package main

import "fmt"

func main() {
    scores := map[string]int{
        "Алексей": 85,
        "Мария":  92,
        "Иван":   78,
    }
    fmt.Println(scores) // Вывод: map[Алексей:85 Иван:78 Мария:92]

    // Добавление/обновление
    scores["Ольга"] = 95
    fmt.Println(scores["Ольга"]) // Вывод: 95

    // Доступ
    if score, ok := scores["Мария"]; ok {
        fmt.Println("Оценка Марии:", score) // Вывод: 92
    }

    // Итерация
    for name, score := range scores {
        fmt.Println(name, ":", score)
    }
}

Разберём код:

  • map[string]int: Карта с ключами-строками и значениями-целыми числами.
  • scores["Ольга"]: Добавление/обновление пары.
  • if score, ok :=: Проверка наличия ключа.
  • for ... range: Перебор пар.

Запуск кода

Для выполнения кода используйте Go:

  1. Создайте файл (slices.go или maps.go).
  2. В терминале выполните: 
    go run slices.go
  3. Убедитесь, что Go установлен (go version).

Практика

Выполните задания:

  1. Создайте срез с названиями 5 стран и выведите их.
  2. Добавьте новую страну и замените вторую.
  3. Создайте карту с 3 именами и их баллами.
  4. Обновите баллы одного человека и добавьте нового.
  5. Напишите программу, выводящую имена и баллы в формате "Имя: Баллы".
  6. Создайте срез чисел и выведите только числа, кратные 3.
  7. Создайте карту с задачами и их приоритетами, затем выведите задачу с наивысшим приоритетом.
